    CodeWars¶ ↑

    This a little role-player game around StarWars theme ! Some “easter eggs” have been left out in the story related to web developpers world. Will you be wise enouth to find them ? History will tell us !

    Install¶ ↑

    # clone the repo and run "bundle" command (you need a working ruby binary installed - this is fully tested for Ruby v2.4.1p111)
git clone git@github.com:davidfabreguette/codewars.git
cd codewars && bundle install

    Launch the game¶ ↑

    Let's get this started !

    ruby start.rb
